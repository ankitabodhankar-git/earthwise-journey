# **App Name**: GreenPath AI

## Core Features:

- Smart Footprint Assessment: A multi-step, interactive questionnaire that evaluates user habits across transportation, energy, water, and waste categories using modern Bootstrap 5 forms.
- Eco-Champion Scoring Engine: A custom logic system that calculates sustainability levels from 'Eco Beginner' to 'Green Champion' based on real-time assessment data.
- Personalized Recommendations Tool: An AI-driven advice tool that reasons through user inputs to suggest specific, actionable reduction strategies for their carbon footprint.
- Eco-Goal Manager: Allows users to select, track, and complete weekly sustainability goals with persistent browser-side state management.
- Impact Dashboard: A centralized visualization hub showing current scores, goal progress summaries, and improvement tips for a holistic view of user impact.
- Local Persistence Layer: Utilizes browser localStorage to ensure all assessment results, user goals, and progress data are saved without requiring a backend database.
- Information & Engagement Modules: Includes professional 'About' and 'Contact' sections to provide project context and environmental awareness.

## Style Guidelines:

- Primary: Emerald Green (#1B7352) for a professional sustainability aesthetic.
- Background: Sage Mist (#F1F7F5) to create a clean, tranquil user environment.
- Accent: Flora Lime (#63B870) used for progress bars, goal checkboxes, and successful interaction feedback.
- Primary font 'Inter' (sans-serif) for high legibility and a contemporary, data-focused appearance.
- Minimalist, thin-line icons representing transportation, energy, and water for a lightweight mobile-friendly footprint.
- A responsive card-based layout using Bootstrap 5, optimized for mobile, tablet, and desktop viewing.
- Subtle transitions and gentle scale effects on interactive elements to enhance the user experience without being distracting.
