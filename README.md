# 🌿 EarthWise Journey

### Small Habits. Smarter Future.

EarthWise Journey is an AI-assisted sustainability awareness platform designed to help individuals understand, track, and improve their environmental impact through smart lifestyle choices, personalized recommendations, and habit tracking.

Built as part of the Global Sustainability Challenge, the platform encourages users to make eco-friendly decisions and adopt sustainable habits through an interactive and engaging experience.

---

# 🚀 Project Overview

Many people want to live sustainably but are unsure which daily actions have the biggest environmental impact.

EarthWise Journey helps users:

* Understand their environmental footprint
* Evaluate daily lifestyle decisions
* Track sustainable habits
* Receive personalized recommendations
* Monitor long-term progress

The platform transforms environmental awareness into actionable and measurable improvements.

---

# ✨ Key Features

## 🌱 Sustainability Assessment

Users complete an assessment based on:

* Transportation habits
* Energy consumption
* Water usage
* Waste management
* Lifestyle choices

The system evaluates responses and generates an overall sustainability score.

---

## 🌍 Smart Choices Engine

Users explore common daily decisions and compare their environmental impact.

Examples:

* Public Transport vs Private Vehicle
* Reusable Bottle vs Single-use Plastic
* Sustainable Products vs Fast Fashion
* Digital Notes vs Printed Notes

This feature helps users make environmentally responsible choices.

---

## 📊 Earth Impact Dashboard

The dashboard provides:

* Sustainability score
* Environmental insights
* Progress tracking
* Habit completion statistics
* Personalized improvement suggestions

---

## 🤖 Personalized Recommendations

The platform generates tailored recommendations based on user responses.

Examples:

* Reduce unnecessary electricity usage
* Use public transportation more frequently
* Reduce plastic consumption
* Improve recycling habits
* Save water through daily actions

---

## 🏆 Habit Tracking System

Users can:

* Track sustainable habits
* Monitor improvement over time
* Build eco-friendly routines
* Increase their sustainability score

---

# 🔒 Privacy & Data Protection

EarthWise Journey follows a privacy-focused approach.

* No user accounts required
* No personal information stored
* Local Storage used for user progress
* User data remains on the device
* No backend database for personal records
